Comentários:


Os Comentários são uma forma de você colocar um texto em seu código para lembrar ou infomar para outros programadores o que você ta fazendo:


Isso é muito útil, principalmente quando você pega um progama que não mexe ha muito tempo, os comentáios vão te ajudar a lembrar o que você fez.


// Esse é um comentário de uma linha

Operadores de Atribuição.

O sinal de igualdade ( = ); serve para atribuir valores a variável.

var idade = 142;


console.log("Sua idade é " + idade + " anos.");

Sua idade é 142 anos.

Atribuimos um valor a variavel idade mas podemos usar qualquer tipo de dado para fazer a atribuição com o sinal de ( = ).

var nome = "theCoyote";


var ativo = true;


var altura = 1.86;


console.log("Boa noite " + nome + ", você tem " + altura + " de altura e sua situação esta ativa = " + ativo);

Boa noite theCoyote, você tem 1.86 de altura e sua situação esta ativa = true


Atribuição de Adição.

Os sinais de Adição( + ), Subtração( - ), Multiplicação( * ), Divisão( / ), Módulo( % ) e Exponenciação( ** ) funcionam da mesma forma que você ta acostumado a fazer na mátematica com exceção do operador Módulo.

O operdor módulo também é conhecido como resto da divisão.

O sinal de + é usado para juntar(concatenar) duas ou mais String's.

O conjunto de sinais mais o operador igual( = ) informam que voce quer somar o valor da variável e salvar nela mesma com o resultado obtido.

var valor = 10;


valor += 2;


console.log(valor);

12      // a variável valor valia 10 e acrescentou ela + 2 e salvou nela mesma, isso sera a mesma coisa de você digitar -> valor = valor + 2 ;

Subtração, Multiplicação e Divisão.

Semelhante ao operador de adição, esses três operadores tem a mesma sintaxe.

var saldo = 500;


var aluguel = 100;


var ajuda = 5


var familia =4


// defini minhas variáveis, agora vou controlar o fluxo de valores em minha carteira.


console.log("O saldo atual é -> " + saldo);


saldo -= aluguel;


console.log("O saldo atual é -> " + saldo);


saldo *= ajuda;


console.log("O saldo atual é -> " + saldo);


saldo /= familia;


console.log("O saldo atual é -> " + saldo);


O Saldo atual é -> 500


O Saldo atual é -> 400


O Saldo atual é -> 2000


O Saldo atual é -> 500


Esse exemplo parece vida de rico, o cara recebe o salário, paga as contas, por algum milagre do ceu tem uma ajuda, divide o lucro com a família e mesmo assim ainda fica com o mesmo valor de quando começou.



Resto da divisão (Modulo %).

Para quem nunca estudou programação esse operador parece bem estranho, por isso vou dedicar um tópico só para ele

Como o nome sugere, esse operador retorna o que sobrou sa divisão

Usando o mesmo exemplo agora com a vida de um probre com a mesma sorte.

var saldo = 500;


var aluguel = 100;


var ajuda = 5


var familia =4


// defini minhas variáveis, agora vou controlar o fluxo de valores em minha carteira.


console.log("O saldo atual é -> " + saldo);


saldo -= aluguel;


console.log("O saldo atual é -> " + saldo);


saldo *= ajuda;


console.log("O saldo atual é -> " + saldo);


saldo %= familia;


console.log("O saldo atual é -> " + saldo);


O Saldo atual é -> 500


O Saldo atual é -> 400


O Saldo atual é -> 2000


O Saldo atual é -> 0


Incremento.

O operador de adiciona 1 a sua variável. Se usado como prefixado (++a), retorna o valor da variável após a adição. se usado pósfixado, retorna o valor da variavel antes da adição.


Exemplo pósfixado.


var a = 8;


var b = 0;


b = a++;


console.log(b);


8

Exemplo prefixado.


var a = 8;


var b = 0;


b = ++a;


console.log(b);


9


Além do encremeto, temos também o de decremento (a--) que subtrai 1 da variável. Sua sintaxe é a mesma do encremento só que subtrai em vêz de adicionar.

Exponencial.

O operador de exponenciação eleva a variável (numerica) a uma potencia.


var quadrado = 8 ** 2;


console.log(quadrado);


64

Operadores de comparação.

A tabela abaixo descreve o que cada operador de comparação faz.


Op.
Descrição
Exemplo
==
Igual retorna verdadeiro se os operandos forem iguais.
5 == "5"
!=
Diferente retorna verdadeiro caso os operandos não sejam iguais
5 != "3"
===
Estritamente igual retorna verdadeiro se os operandos e tipos forem iguais.
5 === 5
!==
Estritamente diferente retorna verdadeiro caso os operandos não sejam iguais e/ou do mesmo tipo.
5 !== "5"
>
Maior que retorna verdadeiro caso o operador da esquerda seja Maior que o da direita.
6 > 5
>=
Maior ou Igual retorna verdadeiro se o operando da esquerda for maior ou igual o operando da direita.
5 >= 5
<
Menor retorna verdadeiro se o operando da esquerda for menor que o operando da direita.
4 < 5
<=
Memor ou Igual retorna verdadeiro se o operando da esquerda for menor ou igual ao operando da direita.
5 <= "5"

=> não é operador, mas uma anotação para função de seta.


=< nunca vi nem ouvi falar. ( isso pode causar um erro no seu código).


Operadores lógicos.

Antes de terminar a primeira fase de operadores, vamos dar uma olhada nos operadores lógicos.

Op.
Descrição
&&
E logico - retorna verdadeiro se todos os operandos sejam verdadeiro.
||
OU lógico - retorna verdadeiro se um dos operandos forem verdadeiro.
!
Não lógico - nega a condição.

Daremos mais explicações sobre como funciona esses operadores nos próximos caípulos, mas, por enquanto só precisamos saber que esses operadores servem para retornar um valor lógico (true ou false) para condições imposta no código que você escreve.
