Funções.

Funções são fragmentos de código que executam uma rotina que são chamadas pelo seu programa toda vez que você precisa

Vamos supor que você tenha um programa para controle de comércio e, neste programa você precisa verificar se os campos dos cadastro de funcionários, fornecedores, produtos e clientes estão sendo preenchidos. Sem uma função, voce teria que repetir a mesma validação para todos os campos em todos os formulários de cadastro, isso definitivamente daria muito trabalho pra fazer além de que toda vez que você precisar mudar alguma coisa nesta rotina voce teria que mudar em todos os lugares que você ela(a rotina).


As funções trouxeram a solução para esse problema. Com ela podemos escrever a rotina uma vez e usa-la sempre que for necessário.


Agora vamos supor que você precise um array em ordem

var clientes = ["Zé", "João", "Maria", "Carlos", "Alberto", "Anderson", "Tatiana"]


function ordenar(lista){


lista.sort();


console.log(lista);


}


ordenar(clientes);


["Alberto", "Anderson", "Carlos", "João", "Maria", "Tatiana", "Zé"]

Vamos entender o que estamos fazendo.

  • Primeiro criamos uma array com o nome de clientes.
  • Iniciamos nossa função, demos o nome dela de ordenar.
  • Informamos que ela iria receber uma lista como parametro ( que esta entre parenteses ).
  • abrimos o bloco de instruções usando a { .
  • Colocamos nosso array em orden usando o sort().
  • Mandamos mostrar o array ordenado no console.
  • E finalmente fechamos o bloco de código usando }.

    • Com essa função, podemos colocar qualquer array que ele vai ordenar.


    var frutas = ["Cajú", "Maçã", "Uva", "Laranja", "Melancia"];


    ordenar(frutas);


    ["Cajú", "Laranja", "Maçã", "Melancia", "Uva"]

    var compras = ["Macarrão", "Feijão", "Arroz", "Açucar", "Café", "Frango", "Carne", "Farinha", "Ovos", "Oleo"];


    ordenar(compras);


    ["Arroz", "Açucar", "Café", "Carne", "Farinha", "Feijão", "Frango", "Macarrão", "Oleo", "Ovos"]

    Quando você usa uma função com parametros, oque você usa no lugar do parametro é substituido, no nosso caso nosso parametro tem o nome lista, quando chamamos a função usando um array com outro nome, internamente o nome lista é substituido pelo nome que usamos em toda função.


    Então quando você ta usando a array com o nome fruta, no lugar de lista o interpretador vai ler o nome fruta, compras ou clientes( que foi o nome que você usou).
