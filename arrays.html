Array

Array são objetos que podem armazenar vários valores.

Imagine que você precise criar 10 variáveis com os nomes dos produtos que você precisa trazer do mercado.

var produto1 = "Macarrão";


var produto2 = "Feijão";


var produto3 = "Arroz";


var produto4 = "Açucar";


var produto5 = "Café";


var produto6 = "Frango";


var produto7 = "Carne";


var produto8 = "Farinha";


var produto9 = "Ovos";


var produto10 = "Oleo";


Com o array, você pode fazer a mesma coisa com menos escrita.

Para declarar um array voce precisa usar a declaração var + nome do array + sinal de igualdade ( = ) + abrir e fechar colchetes ( [] ).

var produto = ["Macarrão", "Feijão", "Arroz", "Açucar", "Café", "Frango", "Carne", "Farinha", "Ovos", "Oleo"];

Arrays são objetos semelhantes a listas que vêm com uma série de métodos embutidos para realizar operações de travessia e mutação. Nem o tamanho de um array JavaScript nem os tipos de elementos são fixos. Já que o tamanho de um array pode ser alterado a qualquer momento e os dados podem ser armazenados em posições não contíguas, arrays JavaScript não tem a garantia de serem densos; isso depende de como o programador escolhe usá-los. De uma maneira geral, essas são características convenientes, mas, se esses recursos não são desejáveis para o seu caso em particular, você pode considerar usar arrays tipados.

    Verificando o tamanho de um array.

    Para verificar o tamanho de um array usamos o .length


    var meuArray = ["Cajú", "Maçã", "Uva", "Laranja", "Melancia"];


    console.log(meuArray.length);


    5 // retorno da quantidade de elementos do array.

    Sua sintaxe é nomeDoArray.length

    Ordenar um array.

    Você pode colocar um array em ordem usando o .sort().


    Sua sintaxe é nomeDoArray.sort().

    var meuArray = ["Cajú", "Maçã", "Uva", "Laranja", "Melancia"];


    console.log(meuArray);// mostrando o array no console


    meuArray.sort();


    console.log(meuArray) // mostrando o array no console -> agora ordenado.


    ["Cajú", "Laranja", "Maçã", "Melancia", "Uva"]

    Adicionando um novo item no final do array.

    Você pode adicionar um item no final do array usando o .push.


    Sua sintaxe é nomeDoArray.push(Aquí dentro vai oque você quer incluir no array.);

    var meuArray = ["Fiat", "Omega", "Gol", "Corsa"];


    meuArray.push("Mercedes");


    console.log(meuArray);


    ["Fiat", "Omega", "Gol", "Corsa", "Mercedes"]


    Você deve sempre incluir o novo item entre os parênteses do push -> meuArray.push("Fusca");. Usando o exemplo do array acima, o meuArray ficaria assim -> ["Fiat", "Omega", "Gol", "Corsa", "Mercedes", "Fusca"]


    O push aceita calculos e concatenação de string. -> meuArray.push(5*3); meuArray.push("Carro"+" "+"de"+ " " + "Mão");


    ["Fiat", "Omega", "Gol", "Corsa", "Mercedes", 15, "Carro de Mão"]


    Remover um item do final do array.

    Para remover o ultimo elemento de um array use o .pop. Sua sintaxe é nomeDoArray.pop().


    Usando o array do nosso ultimo exemplo.


    var meuArray = ["Fiat", "Omega", "Gol", "Corsa", "Mercedes", 15, "Carro de Mão"]


    meuArray.pop


    console.log(meuArray);


    ["Fiat", "Omega", "Gol", "Corsa", "Mercedes", 15]


    Removendo um intem do início do array.

    Use o .shift() para remover um item no início do array.


    Sua sintaxe é nomeDoArray.shift();


    var meuArray = ["Fiat", "Omega", "Gol", "Corsa", "Mercedes", 15];


    meuArray.shift();


    console.log(meuArray);


    ["Omega", "Gol", "Corsa", "Mercedes", 15]


    Adicionando um item no início do array.

    Use o .unshift para adicionar um item no início do array.


    Sua sintaxe é nomeDoArray.unshift(item a ser adicionado.


    var meuArray = ["Omega", "Gol", "Corsa", "Mercedes", 15]


    meuArray.unshift("Montana");


    console.log(meuArray);


    ["Montana", "Omega", "Gol", "Corsa", "Mercedes", 15]


    Procurar o índice pelo item do array.

    Para procurar o índece pelo item do array use o .indexOf.


    Sua sintaxe é nomeDoArray.indexOf(nome do item a ser procurado);


    var meuArray = ["Montana", "Omega", "Gol", "Corsa", "Mercedes", 15];


    meuArray.indexOf("Gol");


    2

    Você pode atribuir o retorno de uma operação a uma variável para que você possa usar ela em uma outra operação.


    var meuArray = var meuArray = ["Montana", "Omega", "Gol", "Corsa", "Mercedes", 15];


    var resposta = meuArray.indexOf("Gol");


    console.log(resposta);


    O resultado desta operação será -> 2


    Remover um ou mais itens pela posição do índece.

    Você pode apagar um ou mais itens de um array usando um índice.


    Sua sintaxe é nomeDoArray.splice(posiçãoInicial, quantidadeDeItens).


    var meuArray = ["Montana", "Omega", "Gol", "Corsa", "Mercedes", 15];


    meuArray.splice(2, 3);


    console.log(meuArray);


    ["Montana", "Omega", 15]

    O splice vai apagar os 3 itens do array começando com a posição inicial.

    Copiar um array.

    Para você copiar um array use .slice().


    Sua sintaxe é var nomeDaCopia = nomeDoArray.slice();

    var meuArray = ["Montana", "Omega", "Gol", "Corsa", "Mercedes", 15];


    var copiaMeuArray = meuArray.slice();


    console.log(copiaMeuArray);

    ["Montana", "Omega", "Gol", "Corsa", "Mercedes", 15]
